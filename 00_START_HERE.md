## 🔧 Firebase Setup (10 minutes)

1. Go to [firebase.google.com](https://firebase.google.com)
2. Click "Create Project"
3. Name it (e.g., "minimal-messaging")
4. Go to Authentication → Enable Google
5. Go to Firestore → Create Database (test mode)
6. Copy your config
7. Update firebaseConfig in your code
